Software:
  • Excel code for C-14 wiggle-match dating (Blaauw et al., 2003; Now uses IntCal04 and cal BP; zipped 2.1 MB), see also the manual (pdf). No longer maintained, please use Bacon instead.
  • Bpeat, software for Bayesian age-modeling of high-resolution C-14 dated peat cores (Blaauw and Christen, 2005a). Zip-file includes manual. (no longer maintained - please use Bacon instead).
  • clam, software for "classical", non-Bayesian age-depth modelling (Blaauw, 2010).
  • Bacon, software for flexible Bayesian age-depth modelling (Blaauw and Christen 2011).
