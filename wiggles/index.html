Software

For more information, please mail me.

R packages:
  • clam, for "classical", non-Bayesian age-depth modelling (Blaauw, 2010)
  • Bacon, for Bayesian age-depth modelling (Blaauw and Christen 2011)
  • rplum, for Bayesian 210Pb age-depth modelling (Aquino et al. 2018)
  • IntCal, a data package containing the IntCal (Reimer et al. 2020) radiocarbon calibration curves, as well as functions to plot them and calibrate/plot radiocarbon dates
  • coffee, for Chronological Ordering for Fossils and Environmental Events. Can be used to produce age-models for trees (with exactly-known time-spans) or stratigraphies (where only the relative chronological order of dates is known)

Old software (no longer maintained):
  • Excel code for C-14 wiggle-match dating (Blaauw et al., 2003; Now uses IntCal04 and cal BP; zipped 2.1 MB), see also the manual (pdf). No longer maintained, please use Bacon instead.
  • Bpeat, software for Bayesian age-modeling of high-resolution C-14 dated peat cores (Blaauw and Christen, 2005a). Zip-file includes manual. No longer maintained - please use Bacon instead.
