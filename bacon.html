Bacon is available as a CRAN R package. Make sure you have installed a recent version of R (at least 4.0.1). Install the package by typing, within the R terminal, install.packages('rbacon') (note the name and the lower case). Choose a mirror near to you. To load the code, type library(rbacon) or require(rbacon).

Please read the tutorials by typing browseVignettes('rbacon') or clicking on:
